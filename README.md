# ng-password-generator
AngularJS directive to generate a password.

    <password-generator field="" password-length="" uppercase="" numbers="" specials=""></password-generator>

## Status
Developing

## Demo
Comming soon

## Options
* field: string
* passwordLength: integer
* uppercase: boolean
* numbers: boolean
* symbols: boolean
